# Ladon - Distributed Consensus Framework based on ISS
Ladon is a distributed consensus framework built on top of the ISS framework. It extends the capabilities of ISS by incorporating the following modifications:

- Addition of Rank Collection Step in the Consensus Protocol: We have introduced a new step in the consensus protocol to collect the ranks of participating nodes. 
- Dynamic Global Ordering Algorithm: We have replaced the existing global ordering algorithm in ISS with our dynamic ordering algorithm. This algorithm optimizes the ordering of transactions to improve overall system performance and efficiency.

Supported Consensus Protocols:
PBFT (Practical Byzantine Fault Tolerance)
HotStuff
